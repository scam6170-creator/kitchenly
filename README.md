# KITCHENLY - Inventory & Sales Management System

A complete professional inventory and sales management web application with modern premium UI.

## Features

- **Authentication**: Secure JWT-based login system with role-based access control
- **Roles**: Administrator, Warehouse Manager, Seller
- **Dashboard**: Real-time analytics and key metrics
- **Products Management**: Create, edit, delete products with categories
- **Stock Management**: Track inventory with low stock alerts
- **Sales Module**: Process sales transactions
- **Reports**: Daily, weekly, monthly, yearly reports with export to Excel/PDF
- **Employee Management**: Manage staff and permissions
- **Global Search**: Search across entire database
- **Responsive Design**: Desktop, tablet, and mobile support
- **Multi-language**: English and Uzbek support
- **Dark/Light Mode**: Theme switching capability

## Tech Stack

### Frontend
- React 18 with TypeScript
- TailwindCSS for styling
- Shadcn UI components

### Backend
- Node.js with Express
- PostgreSQL database
- JWT authentication
- bcrypt password hashing

## Quick Start

```bash
# Backend
cd backend && npm install && npm run db:migrate && npm start

# Frontend
cd frontend && npm install && npm start
```
